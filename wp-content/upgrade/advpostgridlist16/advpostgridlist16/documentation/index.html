Visual Composer Post Layout : Advanced Grid/List Plugin Documentation

We would like to thank you for purchasing Visual Composer Post Layout : Advanced Grid/List Plugin!

This document covers the installation and the usage of this plugin.

if you are experiencing any difficulties or if you have any questions that are beyond the scope of this document, feel free to contact us

System Requirement

Visual Composer Post Layout : Advanced Grid/List plugin supports last version of Wordpress and Visual Composer.

Note: Before use this plugin Install Visual Composer

Plugin Installation

Visual Composer Post Layout : Advanced Grid/List installation doesn't differ from any other plugin installation process,so you might be familiar with this process already. If not, please follow instructions below.

Follow the steps below to install via WordPress.

Step 1 – Login to your website

Step 2 – Go to Plugins -> Add New

Step 3 – Click Upload Link

Step 4 – Browse to the plugin's zip file (it is located in the archive you've downloaded from CodeCanyon) and choose that file.

Step 5 – Click "Install Now" button and wait while plugin is uploaded to your server

Step 6 – Click "Activate Plugin" button

Using Grid/List Post Layout

After Plugin Installation, "Pw Grid" () and "Pw List" () will be added to Visual Composer`s shortcodes. After click, You'll see below fields:

  • Grid Options :
    • Title : Enter your title.which will be display over your shortcode.
    • Build Query : You can choose data source(post,custom post,page,...),number of items and order by. Then you can filter them by category,Tags and ...
      • Choose data source Select post types to populate posts from. Note: If no post type is selected, WordPress will use default "Post" value.
      • Post Count : How many teasers to show? Enter number or word "All".
      • Order By : Select how to sort retrieved posts. More at WordPress codex page
      • Categories : Filter output by posts categories, enter category names here.
      • Tags : Filter output by posts tags, enter tag names here.
      • Taxonomies : Filter output by custom taxonomies categories, enter category names here.
      • Individual Posts/Pages/Custom Post Types : Only entered posts/pages will be included in the output. Note: Works in conjunction with selected "Post types".
      • Author : Filter by author name.
      • Do you need anything else?!
    • Link Target : We provide two types for link target. Same Window: Open link in same window. New Window: Open link in new window.
    • Grid Type : You can choose Grid type. Standard(Fix row),Advanced Grid (Mixitup) and Masonry
    • Count of Columns : There are three fields for column. You can choose count of column for Descktop(PC), Tablet and mobile Size(mode).
    • Grid Skin Type : You can choose Grid Skin Type.
      • Skin one : In this skin you have many fields and option for customize grid item's style.
        • Teaser Layout : Make Visual Layout for Items with drag & drop and generate many combinations..
        • Border Options : You can determine border size,type and color for Top,Right,Bottom and Left
        • Read More Type : Choose "Read more" type.We provide two types. 1- Button mode 2- Link mode
      • Skin two : In this skin you have multi preset styles and effect for Grid`s items
        • Grid Effect : We provide 4 effetcs for Grid`s Items. Test and enjoy!
    • Excerpt/Content Length : If you want to define count of content word, enter number here.
    • Thumbnail size : Enter thumbnail size. Example: thumbnail, medium, large, full or other sizes defined by current theme. Alternatively enter image size in pixels: 200x100 (Width x Height).
    • Image Effect : Choose image effect.
    • Icon Effect : Choose icon effect.
    • Page Navigation : Choose Page navigation type: No Pagination, Display "Show More" button,Infinite Scroll and page number
    • Add Filter : You can choose any custom taxonomies for use in Filter.
    • Default Display Type : You can choose default display type in Advanced Grid.(Grid/List).
    • Show Filter : If "YES" Filter(s) will be displayed.
    • Show Order By : If "YES" Order will be displayed.
    • Hide Switch Icon : Hide icon for switching between Grid and List.
    • Other Color : Choose color for Link ,Link Hover,Meta ,Excerpt,Thumbnail Border and ...
  • List Options:
    • Link Target : We provide two types for link target. Same Window: Open link in same window. New Window: Open link in new window.
    • List Type : We provide two types of list.List with medium image or large image.
    • Excerpt/Content Length : If you want to define count of content word, enter number here.
    • Thumbnail size : Enter thumbnail size. Example: thumbnail, medium, large, full or other sizes defined by current theme. Alternatively enter image size in pixels: 200x100 (Width x Height).
    • Date Format : Enter date format.
    • Image Effect : Choose image effect.
    • Icon Effect : Choose icon effect.
    • Page Navigation : Choose Page navigation type: No Pagination, Display "Show More" button,Infinite Scroll and page number
    • Filter By : If "YES" Filter will be displayed.
    • Order By : If "YES" Order will be displayed.
    • Border Options : You can determine border size,type and color for Top,Right,Bottom and Left
    • Other Color : Choose color for Link ,Link Hover,Meta ,Excerpt,Thumbnail Border and ...
    • Read More Type : We provide two types for read more : type1: Button Style Typle2: Link Style
